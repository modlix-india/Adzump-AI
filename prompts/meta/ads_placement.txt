## Important Rules (Strict)

1. Use ONLY placements from the allowed list
2. Do NOT rename placements
3. Each placement MUST include a short reason
4. Output must strictly follow the required format

---

## Industry → Placement Priority Logic

Interpret priorities as:

- HIGH → primary
- MEDIUM → secondary
- LOW → secondary (lower tier) OR simply omit
- NOT RECOMMENDED → avoid
- AVOID → avoid

---

## Conflict Resolution

- GLOBAL RULES always override INDUSTRY RULES.
- If a global rule forces a placement into primary, remove it from secondary/avoid regardless of industry rule.

## Global Rules

1. If objective = OUTCOME_LEADS AND creative_type = IMAGE:
   - ALWAYS include facebook_feed in primary
   - ALWAYS put reels (instagram_reels, facebook_reels, facebook_instream_reels) in avoid

2. If objective = OUTCOME_AWARENESS AND creative_type = VIDEO:
   - Move facebook_reels, facebook_instream_reels, and instagram_reels to secondary
     (even if industry rules say avoid)

3. If creative_type = CAROUSEL:
   - ALWAYS put stories (facebook_stories, instagram_stories) in avoid
     (Carousel format is not supported in Stories placements)
   - ALWAYS put reels (facebook_reels, facebook_instream_reels, instagram_reels) in avoid
     (Carousel format is not supported in Reels placements)

4. Profile feeds (facebook_profile_feed, instagram_profile_feed) should be placed
   in secondary or avoid — never primary.

5. Explore (instagram_explore, instagram_explore_home) is MEDIUM priority (secondary).
   However, if an Industry Rule explicitly classifies Explore as LOW or AVOID,
   the Industry Rule takes precedence over this global rule.

  (e.g., "Dental Clinic", "SaaS Startup", "D2C Skincare Brand").
  Be specific — use the business summary to go beyond the generic industry label.
- "primary", "secondary", "avoid": Each must contain a list of objects,
  where each object has exactly two fields: "placement" (string) and "reason" (string).
- Every placement from the allowed list must appear in exactly one of: primary, secondary, or avoid.
- No placement should be left unclassified.

  ],

    }}
